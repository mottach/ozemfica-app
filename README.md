# Ozemfica · GLPerfection · GLPerfecto

**Plataforma colaborativa de saúde para a jornada GLP-1 — Guiada por IA**

🔗 **[Abrir o App](https://mottach.github.io/ozemfica-app/)**

---

## O que é

A Ozemfica unifica médicos, nutricionistas, treinadores e pacientes em uma jornada integrada de autocuidado — com IA generativa como copiloto disponível 24/7, especialmente para usuários de GLP-1 (Ozempic, Wegovy, Mounjaro e similares).

## Portais

| Portal | Funcionalidades |
|--------|----------------|
| 🧑‍⚕️ **Médico** | Prescrição, alertas de risco, histórico clínico, análise de fotos corporais, WhatsApp integrado |
| 🥗 **Nutricionista** | Plano alimentar, análise de foto de prato, dispensa virtual, receitas personalizadas |
| 💪 **Treinador** | Plano de treino, exercícios sugeridos por IA a partir de fotos corporais |
| 👤 **Paciente** | Dashboard unificado, diário alimentar, fotos de progresso, achievements |
| 🤖 **AI Coach** | Chat 24/7 com IA, plano nutri + treino gerados por IA — sem profissionais (R$ 49,90/mês) |

## Features

- 📸 **Foto do prato → macros** — IA identifica calorias, proteínas, gorduras e fibras automaticamente
- 🛒 **Dispensa Virtual** — foto das compras → alimentos catalogados; receitas com o que tem em casa
- 🩻 **Fotos corporais 4 ângulos** — análise de postura, biotipo, equilíbrio muscular por IA
- 💬 **WhatsApp integrado** — mensagem do profissional registrada no prontuário clínico
- 📋 **Histórico clínico unificado** — timeline cross-portal de todas as interações
- 🤖 **AI Coach 24/7** — único no Brasil com jornada GLP-1 completa guiada só por IA

## Stack

- Pure static HTML + CSS + JavaScript (single file, zero build)
- Chart.js para visualizações
- GitHub Pages para hospedagem
